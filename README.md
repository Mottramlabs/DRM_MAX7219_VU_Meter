# DRM_MAX7219_VU_Meter
LED VU Meter usind a MAX7219 LED driver and an Arduino/ESP8266

Pin connections
Name      Arduino     MAX7219 7 Segment display
-----     -------     -------------------------
          +5V         VCC
          GND         GND
DIN       D11/MOSI    DIN
CS        D10/SS      CS
CLK       D13/SCK     CLK
